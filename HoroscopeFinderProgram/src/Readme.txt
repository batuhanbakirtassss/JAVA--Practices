HoroscopeFinderProgram
The HoroscopeFinderProgram is a Java program that allows users to enter their birth month and day, and outputs their corresponding horoscope sign. The program uses a series of if-else statements to determine the user's horoscope sign based on their birth date.

How to Use
To use the program, simply run the program and enter your birth month and day when prompted. The program will then output your corresponding horoscope sign.

Compatibility
The program is written in Java.
