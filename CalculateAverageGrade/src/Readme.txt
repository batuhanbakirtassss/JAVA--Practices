Calculate Average Grade
This Java program allows you to input grades for various subjects and calculates the average grade along with your passing status.

Description
The CalculateAverageGrade program prompts the user to input grades for subjects like Math, Physics, Chemistry, Turkish, History, and Music. It then calculates the average grade and determines whether you have passed or failed based on a passing threshold of 60. The program displays both the average grade and the passing status accordingly.
